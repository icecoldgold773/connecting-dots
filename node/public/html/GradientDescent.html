What is Gradient Descent?

The act of optimizing the neural network would be to minimize error, which are part of the cost function, leading to a maximized accuracy for the algorithm. This can be achieved through changes in the weights and biases connected to each neuron in the neural network.


The general idea of gradient descent is to find a set of weights and biases within the cost function such that the local minimum is reached, meaning that the error is minimal. This is done using multivariable calculus to find the derivative or the slope of the function and then changing the weigths and biases slighty in the direction of the local minimum. This action is then repeated over and over, until the changes become inconsequential, and a local minimum is reached. This is done in backpropagation, read about backpropagation here
