What are Neural Networks?

The neural network (NN) is a type of machine learning technique inspired by the biological neural network - The brains network. The biological neural network consists of a network of neurons. Every neuron obtain electric signals from other neurons and then they treat this information and sends it to other neurons. Its the same principle in the artificial neural network, as it has artificial neurons.


Another crucial concept to neural network is weights and biases, which are parameters in the neural network that simplify data identification


Weight refer to a connection between two neurons


Bias is added to a neuron to help influence the end product


The artificial neural network works by taking one or more inputs, applying different functions and weights and then turning this into one or more outputs.

The artificial neural network consists of one input layer one or more hidden layers and lastly one output layer. This type of neural network is also called a feed forward neural network because the connection between the neurons doesn't create a cycle. But there exists various different architectures, such as feedforward, recurrent, and convolutional networks, to model various types of data and tasks, such as image recognition, speech recognition, natural language processing, and decision-making. Convulutional neural network

CNN is a type of feed-forward NN that classifies images by detecting different features in an image. A CNN is generally made up of three types of layers: Convolutional layers, pooling layers, and fully connected layers. Depending on the complexity of the classification problem, there can be multiple convolutional layers and pooling layers, but there is only one fully connected layer.

In the convolution layer patterns in the image are detected. The convolution layer consists of several filters (also referred to as kernels). Each filter is a matrix of weights that is applied by sliding the filter over the image and then taking the weighted sum of the pixel. The value of each weight in a filter is typically initialized randomly, and then adjusted during the training phase (backpropagation) to detect the important features to correctly classify the image, read about backpropagation here


After the convolution layer, the output is passed through an activation function which is typically the Rectified Linear Unit (ReLu) function, read about activation functions here


Same padding is the act of adding a border of pixels around the image with a value of zero. This is done to ensure that the output of the convolution layer has the same dimensions as the input image.


The pooling layer is used to reduce the size of the image, and hereby reduce the number of parameters in the network. This helps reduce the computational complexity of the network, without losing too much information.


The fully connected layer is the last in CNNs. The fully connected layer is a standard NN layer and is the final step in the classification process.
