What are Cost functions?

A cost function is used for measuring the difference between the expected output of our neural network and the actual output. The cost function does this by taking the actual and expected outputs as inputs, and returns a single number thats indicative of how close the actual output was to the expected output.


There are a lot of different cost functions that can be used, however there are two loss functions that are used more often than the rest. These two are Mean Squared Error (MSE) and Cross Entropy (CE)


The Mean Squared Error function finds the difference between the expected output and the actual output, and squares this number. If there are multiple outputs, this function finds the average squared difference. Since the difference between the 2 outputs are squared, this number will always be positive which makes it easier to work with later on during the backpropagation, read about backpropagation here


CE or "Logarithimic cost" is a cost function that is commonly used in classification problems. This cost function is ideal for classification problems as it always returns a value between 0 and 1, where the output represents the chance of the input belonging to a class. This function is run once for all classes, giving the probability of the input belonging to each class. The ideal output for this function is 1 for the correct class, and 0 for all others
