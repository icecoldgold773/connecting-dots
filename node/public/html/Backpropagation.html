What is Backpropagation?

Backpropagation is a way of computing the gradient descent of the cost function by systematically using the chainrule. What the algorithm does, is that, after each forward pass though the network it performs a backwards pass while adjusting the models parameters (weights and biases). The level of adjustment is determined by the gradients of the cost function with respect to those parameters.


The reason for moving backwards through the network is that the cost function is a function of the outputs from the network. To understand how the cost varies with earlier weights and biases it is essential to start at the last layer of the network and by repeatedly using the chain rule the backpropagation algorithm can work its way backwards through the layer and obtain useful expressions


The chain rule is differentiation of a composite function. You differentiate a composite function by differentiating the outer function with the inner function untouched, and then multiply it by the inner function differentiated.
