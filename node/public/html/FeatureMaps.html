What are Feature Maps?

The convolution of an image happens in the convulutional layer of a convulutional neural network (CNN), read about CNN here. This can be visualized with a feature map, which is just the output matrices of the convolution layer drawn as images, where the value of each matrix entry determines the brightness of the corresponding pixel. The higher the value of an entry in the matrix, the brighter the corresponding pixel will often be displayed. A feature map can help visualize what features the model is detecting.
